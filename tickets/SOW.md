# SOW-007

## Title

Create the `browser` skill.

## Objective

Deliver an isolated Developer Dashboard skill that exposes Playwright-backed CLI browser automation through `dashboard browser.get`, `dashboard browser.post`, and `dashboard browser.png`.

## Deliverables

- isolated skill repository with versioning and changelog files
- dependency files for DD skill installation on Debian-family and macOS hosts
- Playwright-backed GET and POST CLI commands
- Playwright-backed screenshot capture through `dashboard browser.png`
- Docker-based tests with `100%` coverage
- updated docs, release commit, and release push
